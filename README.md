# AimCv

AimCv is an experimental computer vision project focused on camera-based aim tracking and visual feedback.

The goal is to build a computer-vision-based hunting simulator that computes an aim direction from camera input, estimates intersections in 3D space, and visualizes the result in real time using OpenCV.

Development notes and progress updates are available at:
[https://antsur1.github.io/AimCv/](https://antsur1.github.io/AimCv/)
